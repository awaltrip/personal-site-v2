Nora is Nationwide.com's enterprise self-service AI chatbot. A back-end Node.js (TypeScript) application built with 
the Microsoft <a href="https://dev.botframework.com/" target="_blank" rel="noopener noreferrer">Bot Framework</a> SDK, 
utilizing Microsoft's <a href="https://luis.ai" target="_blank" rel="noopener noreferrer">"LUIS"</a> conversational AI, 
and hosted in Azure cloud. Project also involved building new REST APIs and integration of Nora into front-end web 
applications (Angular & Spring MVC).<br/><br/>
